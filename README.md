# Logistics Tracker

A centralized, rule-based fleet management system built to replace manual logbooks. It helps teams track the full delivery lifecycle, monitor driver safety, and keep an eye on fleet and financial performance.

## What it covers

- Fleet and vehicle management
- Driver records and safety tracking
- Basic operational and financial visibility

## Scripts

- `npm run dev` - start the app in watch mode
- `npm run start` - start the app
- `npm run typecheck` - run TypeScript checks
